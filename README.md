# motion-detection-system-arduino
Arduino-based motion detection system using a PIR sensor to detect movement and trigger an LED/alarm. Simulated in Wokwi.

# Motion Detection System using Arduino

## Description
This project uses an Arduino Uno and a PIR motion sensor to detect motion. When motion is detected, an LED (or buzzer) is activated and a message is displayed on the Serial Monitor.

## Components
- Arduino Uno
- PIR Motion Sensor
- LED
- 220Ω Resistor
- Jumper Wires

## Software
- Arduino IDE
- Wokwi Simulator

## Working
- PIR sensor detects motion.
- Arduino reads the sensor output.
- LED turns ON when motion is detected.
- LED turns OFF when no motion is detected.
